# Jabalpur Folium Map

This is a small beginner-friendly Python project that plots geospatial data on an interactive map using Folium.

## Files

- `locations.csv` contains the dataset with `name`, `latitude`, and `longitude`
- `main.py` reads the dataset and creates the map
- `map.html` is an included browser preview file

## How to run

1. Install Python
2. Install the dependency:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the script:

   ```bash
   python main.py
   ```

4. Open `map.html` in a browser

Running `main.py` will recreate `map.html` using Folium.
